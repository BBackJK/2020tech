# CSS3 속성

## 박스 모델

### 박스 모델의 개념

웹 문서에 텍스트, 이미지, 테이블 등의 요소를 배치하기 위해 사용

웹 문서의 전체 레이아웃을 정의

각종 요소들을 원하는 위치에 배치

- **박스의 속성**

content: 실체 내용이 표현되는 곳  
padding: 콘텐츠와 테두리 사이의 여백  
border: 박스의 테두리 두께  
margin: 테두리와 박스의 최종 경계 사이의 여백
